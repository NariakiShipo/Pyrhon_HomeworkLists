# 033 有效字串

## 題目說明
一個字串的子字串 (substring) 為該字串的一段連續區間所形成的字串。給定一個字串 exist (5 >= exist 長度 >= 1)，由英文字母與數字組成，且每個字元不重複。再給定三個整數 n, m, condition，以及一個在 exist 中存在的字元 c。
若一個長度為 n 的字串 data，所有字元都存在於 exist 中，且每個長度為 m 的子字串都至少包含 condition 個字元 c，則 data 被稱為有效字串。

## 【輸入說明】
1. 輸入一個 exist 字串，5 >= exist 長度 >= 1。
2. 輸入整數 n，8 >= n >= 1。
3. 輸入整數 m，n >= m >= condition。
4. 輸入整數 condition，m >= condition。
5. 輸入字元 c，c 必須存在於 exist 字串中。

### 範例輸入說明:
	Sample Input:
	abc
	3
	2
	1
	b

## 【輸出說明】
輸出長度為 n 的有效字串總數量。

### 範例輸出說明:
	Sample Output:
	11 (aba abb abc bab bba bbb bbc bcb cba cbb cbc，共 11 個字串符合條件)
