# 037 理想大學

## 題目說明
每一大學可以擁有 8 種屬性：
GF(Good Food)：代表附近有美食。
BC(Big Campus)：代表有大校園。
NC(Next to City)：代表鄰近有大城市。
CT(Convenient Transportation)：代表交通方便。
NS(Next to Sea)：代表靠海。
NM(Next to Mountain)：代表依山。
HL(Has Lake)：代表校園有湖。
NL(Near Landscape)：代表附近有風景區。

## 【輸入說明】
1. 第一行輸入一個正整數 n，代表大學個數 (n ≤ 10)。
2. 接下來 n 行，每行第一項為大學名稱，接著為其屬性（名稱最多 10 字母，各屬性 2 字母，空白分隔）。
3. 再輸入一個正整數 m，代表查詢個數 (m ≤ 10)。
4. 接下來 m 行，每行為一組查詢條件（屬性以空白分隔，+ 號區隔代表「或」，否則為「且」）。
5. 最後輸入一個正整數 b，b=0 或 b=1，決定輸出格式。

## 【限制條件】
1. 查詢條件格式如：BC NS + CT HL，代表【大校園且靠海】或【交通方便且校園有湖】。
2. b=0：輸出最符合條件的大學（符合最多組條件，若多間則依輸入順序）。
3. b=1：輸出部分符合條件的大學（依符合屬性數量由大到小，若同名次則依輸入順序）。
4. 不需考慮沒有任何符合條件的大學情況。

### 範例輸入說明:
#### Sample Input:
	3
	NSYSU NC CT NS NM
	NTU BC NC CT NS
	NCCU BC NL HL
	2
	BC NS + CT HL
	BC + NS
	0

## 【輸出說明】
b=0：輸出最符合條件的大學名稱（多間以空白分隔）。
	NTU

### 範例輸出說明:
#### Sample Output:
	NTU
	NTU

### 【測試資料一】
#### 輸入：
	3
	NCUE GF NL NM
	NSYSU NS
	NCCU CT BC NM HL NC
	2
	HL NL + CT BC + NM GF
	NL + NL CT HL
	0
#### 輸出：
	NCUE NCCU
	NCUE

### 【測試資料二】
#### 輸入：
	4

	HL

	0

	CCU NCNU
	NTPU NUK
	NTPU

	2
	GF NM CT

	1
