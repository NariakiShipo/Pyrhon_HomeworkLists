
# 038 基因序列

## 題目說明
DNA 序列是由大寫英文字母組成的字串，基因 (gene) 是隱藏於 DNA 序列中的連續切片字串。給定一個 DNA 序列字串 L，找出在 DNA 序列字串中的所有基因（可能有多個）。
1. 給定一個前面序列字串以及三個後面序列字串，基因需為被前面序列字串與後面序列字串夾在中間的字串。
2. 基因長度需為質數（不包含前後序列字串）。
3. 基因只會包含 A、T、C、G 四個字元，前面序列字串中除了開頭或結尾，中間可能包含 0 個或更多的「?」字元，「?」可為任何大寫英文字母。

## 【輸入說明】
1. 第一行：輸入一個基因前面序列字串 S。（3 ≤ S.length ≤ 10）
2. 第二行：輸入三個基因後面序列字串 E1、E2、E3（空白分隔，1 ≤ Ei.length ≤ 10）。
3. 第三行：輸入一個 DNA 字串 L。（3 ≤ L.length ≤ 50）

### 範例輸入說明:
#### Sample Input:
    A?G
    TAG TAA TGA
    ATGTTTTAA

## 【輸出說明】
輸出時以基因長度排序，若長度相同則依照字母順序。
若沒有找到基因，則輸出：No gene。

### 範例輸出說明:
#### Sample Output:
	TTT

### 【測試資料一】
#### 輸入：
	AEG
	TAG TGB EWQ
	AEGTTTEWQ
#### 輸出：
	TTT

### 【測試資料二】
#### 輸入：
	AXND
	JYU KUO POI
	IUAXNDTCGAKUOTEAXNDAVGPOI
#### 輸出：
	No gene

### 【測試資料三】
#### 輸入：
	A?C
	EDC RFV TGB
	APCGTACGTGBCKARCGTACTGBAMCTAGEDC
#### 輸出：
	TAG
	GTACG
