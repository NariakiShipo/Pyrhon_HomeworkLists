# 030 字串切片與串接解譯器
## 設計一個程式，模擬解譯器並根據字串的賦值、串接以及切片規則，最終依照輸出指令的參數輸出結果。

## 【輸入說明】
輸入包含兩類資料，「字串變數初始化」、「字串運算與輸出指令」。
1. 第一行輸入整數 M, N。M 是「字串變數初始化」的行之數量，N 是「字串運算與輸出指令」的行之數量。
2. 其後 M 行, 每一行為「字串變數初始化」變數名稱為英文字母，字串內容不含空白或標點符號，單行內可以多個字串變數初始化。
3. 接續 N 行，每一行為「字串運算與輸出指令」，字串運算只包含運算子(+)結合字，單行只有一個運算子。輸出指令參數包含(變數、變數[start:end:step]、變數[start:end] 、變數[index])。

## 【限制條件】
1. 1 <= M, N <= 10
2. 1 <= 單行字串變數初始化數量 <= 10
3. 1 <= 輸出指令數量 <= 10
4. 測試資料皆無錯誤情形
5. 禁止使用eval()

## 範例輸入說明:
    Sample Input :
    2 5
    x, y = 'abc', 'defghik'
    z = 'abc'
    print(y[1:4:2])
    a = x + z
    print(a)
    print(a[:])
    print(z[2:-3:-1])

## 【輸出說明】
    依輸出指令順序與其參數輸出結果。
    範例輸出說明:
    Sample Output :
    eg # print(y[1:4:2])
    abcabc # print(a)
    abcabc # print(a[:])
    cb # print(z[2:-3:-1])

### 【測試資料一】
    輸入:
    2 2
    X = 'OldValue'
    X = 'NewValue'
    print(X)
    print(X[0:3])
    輸出:
    NewValue
    New
### 【測試資料二】
    輸入:
    1 5
    A = 'Go'

    A = A + A
    print(A)
    輸出:
    Go
