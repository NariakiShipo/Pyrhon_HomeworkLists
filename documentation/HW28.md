# 028 疫情分析

## 某個城市的人口共有 people 人，計算期間共有 day 天。

## 規則說明
- 從第1天開始，城市中有 first_confirmed 個人確診。
- 每位確診者在染疫期間會傳染給 infect 個人。
- 確診者經過 rehabilitation 天後會康復。
- 確診中和確診過和天生有免疫力的人不會把疫情傳染給其他人。
- 假設康復天數是7天，第1天確診的人，在第1~7天皆視為確診，在第8天時才視為康復。
- 這個城市第一天有 immunity 比例的人具有對病毒的免疫力，所以people*(1-immunity)個人是可能被傳染的。
- 隨著確診者康復獲得免疫力，immunity會逐漸提高。
- 每名確診者每日能夠傳染的人數 x = (infect / rehabilitation)* (1–immunity)。
- 每天的新增確診數y = 昨日總確診人數 * x，y需無條件捨去小數，只取整數。
- 若當日有人康復，則在計算完當日的新增確診人數後，再進行更新immunity 的計算。
- 每日傳染人數不得超過可被感染的人數(若計算超過則等於可被感染人數)，例如：城市總人口數共為300人，原先有210人具有免疫力不會被傳染，第一天多了90位確診，則城市中「可被感染人數」更新為 300 - (210 + 90) = 0人。代表日後皆不會有新的確診者。

請計算出這個城市每日的總確診人數、新增確診人數、康復人數，並在最後計算期間共有多少人確診。

## 【輸入說明】
1. 第1行：輸入城市人口people，people為整數，100 <= people <= 10000
2. 第2行：輸入計算期間day，day為整數，1 <= day <= 20
3. 第3行：輸入第一天的確診人數first_confirmed，first_confirmed為整數，1 <= first_confirmed <= people * (1 - immunity)
4. 第4行：輸入每位確診者在確診期間的傳染人數infect，infect為浮點數，0.1 <= infect <= 2
5. 第5行：輸入康復天數rehabilitation，rehabilitation為整數，1 <= rehabilitation <= 20
6. 第6行：輸入第一天城市的免疫率immunity，immunity為浮點數，0 <= immunity <= 1

### 範例輸入說明:
    Sample Input :
    1000 // 城市人口共有1000人
    10 // 計算期間共10天
    100 // 第1天城市中有100人確診
    1.2 // 確診者在染疫期間傳染給1.2個人
    7 // 確診者需7天才會康復
    0.7 // 城市有70%的人已免疫
