# 034 列車載重分配

## 題目說明
將貨物裝載到一列每節完全相同的車廂上。為了確保列車在高速行駛時的安全穩定性與平衡性，每節車廂的最終載重必須相等。同時，所有貨物必須全部裝載，不得有剩餘。

## 【輸入說明】
1. 第一行輸入大寫字母 S 字串，1 <= S.length <= 50。
2. 第二行輸入整數 N，表示車廂數量，1 <= N <= 50。
3. 第三行輸入整數 M，表示每節車廂貨物總數不超過此數，1 <= M <= 50。

## 【限制條件】
1. 使用遞迴技巧進行作答。
2. 字母 A 代表貨物重量為 1，字母 B 代表貨物重量為 2，以此類推。
3. 判斷是否存在一種貨物分配方案，能夠將字串 S 中的所有貨物，剛好分配到 N 節車廂中，使得每節車廂的總載重完全一致，並且每節車廂的貨物數量不超過 M。

### 範例輸入說明:
	Sample Input:
	ABCADED
	4
	2

## 【輸出說明】
1. 未分配的貨物數量超出範圍，輸出 error。
2. 輸入的貨物字母不在 A-Z 中，輸出 error。
3. 如果成功分配，輸出 ok。
4. 如果無法分配，輸出 fail。

### 範例輸出說明:
	Sample Output:
	ok # [A,D] [B,C] [A,D] [E] 且數量不超過 2

### 【測試資料一】
