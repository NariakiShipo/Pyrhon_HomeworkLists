# 029文字編輯器

## 設計一個命令列簡易文字編輯器,大小寫視為不同。

## 編輯指令與意義
- `awf i n word`：在第 i 行中，第 n 個字前面加上 word
- `awa i n word`：在第 i 行中，第 n 個字後面加上 word
- `asf i sentence`：在第 i 行前面加入一句 sentence
- `asa i sentence`：在第 i 行後面加入一句 sentence
- `if key word`：在文章中所有 key 前面加上 word
- `ia key word`：在文章中所有 key 的後面加上 word
- `dw i n`：刪除第 i 行中，第 n 個字
- `dl i`：刪除第 i 行
- `rp old new`：將文章中所有 old 替換為 new
- `c`：目前文章總計字數


## 【輸入說明】
1. 第一行輸入兩個整數 M, N, M是原始文章行數,N是編輯指令數。
2. 其後 M 行,每一行為原始文章的一個句子,以空白隔開每個字。原始文章不含標點符號;第幾行、第幾個字都是從 1 開始計算。
3. 其後 N 行,每一行的第一個字串為編輯指令,隨後為對應的控制字串,以空白隔開。

### 範例輸入說明:
    Sample Input :
    2 5 // M=2 (原始文章2行),N=5 (5個編輯指令)
    Hello good morning // 第1行
    This is my World // 第2行
    awa 1 2 amazing // 指令 1: 在第1行第2個字後面加 "amazing"
    rp World earth // 指令 3: 將文章中所有的 "World" 替換為 "earth"
    if is New // 指令 4: 在文章中所有 "is" 前面加上 "New"
    c // 指令 5: 計算目前文章總字數
### 【輸出說明】
    輸出編輯完成之文章。
    範例輸出說明:
    Sample Output:
    This New is my earth
    5

### 【測試資料一】
#### 輸入:
