
# 031 括號檢查

## 題目說明
給定 N 個字串，每個字串由括號、字母、數字和運算符號組成。需檢查括號是否成對出現，以及是否符合括號層級，最後輸出字串運算結果或錯誤訊息。

## 【輸入說明】
1. 第一行輸入一個整數 N，1 <= N <= 5。
2. 接續 N 行，每行為一個待檢查字串 L (0 <= len(L) <= 50)。
3. 成對括號包含三種：大括號 {}、中括號 []、小括號 ()。
4. 括號層級：{} > [] > ()，即小括號內不能包含中括號，中括號內不能包含大括號。
5. 字串運算只在小括號內出現，包含 (*) 複製、(+) 結合，且最多兩字串運算，括號間不運算（如 ha+ha -> haha）。

### 範例輸入說明:
	Sample Input:
	3
	{[((fg*3)(hi+hi)())]}
	{({(Pa+ss)})}
	{}[{ntut]

## 【輸出說明】
1. 括號成對且符合層級，輸出字串運算後結果。
2. 括號不成對，輸出 "Unbalanced"。
3. 括號不符合層級，輸出 "Hierarchy Violation"。
4. 括號不成對且不符合層級，輸出 "Unbalanced"。

### 範例輸出說明:
	Sample Output:
	fgfgfghihi # 成對且符合層級，輸出小括號內運算後字串，其餘字串直接輸出。
	Hierarchy Violation # 括號成對但不符合層級
	Unbalanced # 括號不成對也不符合層級

### 【測試資料一】
#### 輸入：
	4
	abc
	{[(Lo)ng]}
	{{}((P*2))PP}
	{[(P)]}[(a)](ss)
#### 輸出：
	abc
	Long
	PPPP
	Pass

### 【測試資料二】
#### 輸入：
	4
	{({(Python)()})}
	{a({(f)(g)}h)}
	{a[{(f+c)(g+s)}hd]}
	([xy]{(a+b)*((a+c)*3)})
#### 輸出：
	Hierarchy Violation
	Hierarchy Violation
	Hierarchy Violation
	Hierarchy Violation

### 【測試資料三】
#### 輸入：
	4
	{({)(})}
	([xy]{(aq+bf)[(ac]d)})
	()][((xy(ab])[(a+c)d))
	{(h{ab)(xy}z)}
#### 輸出：
	Unbalanced
	Unbalanced
	Unbalanced
	Unbalanced

### 【測試資料四】
#### 輸入：
	5
	{[[(2*YY)(MM+dd)]]}
	{a({(f*5)(g)}h)}
	{}[{]
	[{}](a+b)
	({(a+b)(c+d)})
#### 輸出：
	YYYYMMdd
	Hierarchy Violation
	Unbalanced
	Hierarchy Violation
	Hierarchy Violation
