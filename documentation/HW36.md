
# 036 移除最少數量的括號

## 題目說明
輸入一個由小括號()和英文小寫字母組成的字串 (1 ≤ 字串長度 ≤ 25，0 ≤ 括號數量 ≤ 20)，請從該字串中移除最少數量的小括號，使其符合括號規則。
請輸出需要移除的小括號最少數量，並輸出所有可能的結果（字典序小到大，重複只輸出一次）。

## 【輸入說明】
1. 一行輸入字串。

## 【限制條件】
1. 字典序比較：兩字串從第一個字元開始比較，遇到不同字元時，依字元大小排序。
2. 字元大小順序： ( < ) < a < b < ... < z。

### 範例輸入說明:
	Sample Input:
	()())()

## 【輸出說明】
輸出最少需要移除的小括號數量，並列出所有可能的結果（字典序小到大，重複只輸出一次）。

### 範例輸出說明:
	Sample Output:
	1
	(())() ()()()

### 【測試資料一】
#### 輸入：
	(a)())()
#### 輸出：
	1
	(a())() (a)()()

### 【測試資料二】
#### 輸入：
	((ab))(hy)())(o
#### 輸出：
	2
	((ab)(hy)())o ((ab))(hy())o ((ab))(hy)()o

### 【測試資料三】
#### 輸入：
	((ab()((uu()
#### 輸出：
	4
	((ab)uu) (ab()uu) (ab)(uu) (ab)uu() ab()(uu) ab()uu()

### 【測試資料四】
#### 輸入：
	ag()))(()gj()
#### 輸出：
	3
	ag()(()gj) ag()()gj()
