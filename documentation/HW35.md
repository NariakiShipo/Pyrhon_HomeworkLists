# 035 修復計算機

## 題目說明
輸入一長串連在一起的數字字串（例如 "123"），以及一個目標數值。請找出要在這些數字之間插入什麼運算符號（+, -, *），才能算出這個目標數值？請輸出有幾種方式可以達到目標數值。

## 【輸入說明】
1. 第一行輸入數字字串，8 >= 字串長度 >= 1。
2. 第二行輸入目標數值（總合數值）。

## 【限制條件】
1. 可使用的符號：+ (加), - (減), * (乘)。
2. 輸入 "123"，你可以把它視為 1, 2, 3，也可以視為 12, 3 或 1, 23。
3. 乘法 * 優先於加減法。
4. 數字不能有不合理的前導零，例如 1 + 05 是不合法的（但在字串單獨為 "0" 時允許，如 1 + 0 + 5）。

### 範例輸入說明:
	Sample Input:
	123
	6

## 【輸出說明】
輸出有幾種組合可以算出目標值。

### 範例輸出說明:
	Sample Output:
	2 (1*2*3, 1+2+3，總共有 2 種組合可以算出目標值)

### 【測試資料一】
#### 輸入：
	12345678
	21
#### 輸出：
	29

### 【測試資料二】
#### 輸入：
	99999999
	36
#### 輸出：
	154

### 【測試資料三】
#### 輸入：
	12131415
	17
#### 輸出：
	154
