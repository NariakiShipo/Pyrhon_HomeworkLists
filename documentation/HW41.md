# 041 成績處理

## 題目說明
學校系統的資料庫儲存各項課程資訊，其功能分成兩部分，說明如下：
1.輸出各科系、各年級的每一學年所有課程學生的學年平均成績小於60分中排名前三名的學生學號、學年平均成績、課程名次百分比和退選百分比。
2.輸出每項課程每年的最高學期成績、平均學期成績、最低學期成績和退選百分比，以及課程成績大於40分(不包含40分)的人數。

每個課程資訊包含課程編碼、學期編碼、學生學號編碼和學生成績。各項編碼資訊如下：
課程編碼：共四碼，前三碼是學科編碼(英文一為101、國文一為201)、後一碼是學分數
學期編碼：共四碼，前三碼是學年度，後一碼是學期(第一學期為1、第二學期為2)
學生學號：共八碼，前三碼是入學年度、中間三碼為科系編碼、後兩碼是座號

每個學科一學年僅開一次，且每位學生一年至少上完一堂課，各項數值計算如下：
1.排名：成績較高排名較前，若成績相同則以學號編碼較小排名較前
2.課程退選百分比：課程退選人數/課程總人數，無條件捨去至整數
3.學生退選百分比：學生退選的課程數/學生選修的課程數，無條件捨去至整數
4.課程成績：英文一、國文一學期成績算法為學期成績*0.7+會考成績*0.3，無條件進位至整數，其餘課程成績即學期成績
5.學生學年平均成績：該學年度所有課程的平均成績，(該學年所有未退選課程成績*該課程學分數加總)/該學年所有未退選學分數，無條件捨去至整數
6.一堂課的平均課程成績:(該堂課的所有未退選學生的課程成績總合)/(該堂課未退選的學生總人數)，無條件捨去至整數
7.課程名次百分比：假如課程總人數(包含退選人數有50人)
百分比為1%的人數=(50*0.01無條件進位至整數)=1
百分比為2%的人數=(50*0.02無條件進位至整數)–(百分比為1%的人數)=1-1=0
百分比為3%的人數=(50*0.03無條件進位至整數)–(百分比為1%~2%的人數)=2-1=1
因此排名第一名百分比為1%，排名第二名百分比為3%

請輸入多項課程資訊後，使用dictionary儲存資料，並依序執行所有功能

## 【輸入說明】
1. 第一行：輸入課程數M
以下課程資料輸入重複M次
{
輸入課程資料(課程編碼，學期編碼，學生數量n)，中間以空白間隔
輸入n筆修課學生資料(學號，學期成績，會考成績) ，中間以空白間隔
}
2. 學生資料說明：若學生退選，學期成績為w
非英文一101及國文一201的課程，僅包含學期成績(無會考成績)

## 【輸出說明】
輸出分為以下兩部分，每一部分中間無間隔

1. 第一部分：
輸出所有科系編碼、入學年度、課程學年度，皆由小到大排序，中間以空白間隔
接著輸出所有符合此科系及入學年度的學生中，此學年度學年平均成績小於60分中排名前三名的學生學號、平均成績、課程名次百分比和學生退選百分比，中間以空白間隔。(若未達60分的學生未達三人，則只需要輸出Not Enough Student)

2. 第二部分：
輸出所有課程編碼、課程學年度，皆由小到大排序，中間以空白間隔
下一行輸出此課程該年度所有未退選學生中最高課程成績、平均課程成績、最低課程成績、課程退選百分比以及此課程學生大於40分(不包含40分)的人數，中間以空白間隔。
