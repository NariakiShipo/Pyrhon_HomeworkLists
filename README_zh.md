# Pyrhon_HomeworkLists

## 專案簡介
本專案收錄多份 Python 作業題目與解答，並附有詳細題目說明。

- `docs/` ：存放所有作業題目說明（Markdown 格式），每份檔案（如 HW30.md）包含題目、輸入輸出格式、範例測資。
- `src/`  ：存放各題 Python 解答程式，每份檔案（如 HW30.py）對應一份題目。

## 如何使用本專案

1. **瀏覽題目**
   - 前往 `docs/` 資料夾查看作業題目與需求。
   - 每份 Markdown 檔案命名為 `HWxx.md`（xx 為題號）。

2. **查找解答**
   - 前往 `src/` 資料夾查找各題 Python 解答。
   - 每份 Python 檔案命名為 `HWxx.py`，與 `docs/` 對應題目相同。

3. **執行程式**
   - 在專案根目錄開啟終端機。
   - 執行解答程式：
     ```sh
     python src/HWxx.py
     ```
   - 輸入方式請參考對應 `docs/HWxx.md` 檔案。

4. **測試與除錯**
   - 可使用各題 Markdown 檔案中的範例測資進行測試。
   - 若需修改或除錯，請編輯 `src/` 內的 Python 檔案。

## 範例流程

1. 閱讀 `docs/HW31.md` 題目說明。
2. 查看 `src/HW31.py` 解答。
3. 執行程式：
   ```sh
   python src/HW31.py
   ```
4. 輸入範例資料。
